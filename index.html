WIP: YOLOv8 Fine-Tuning For Cat Pose Estimation

This project adapts a fast, lightweight tracking model (YOLOv8 Nano), originally designed for human poses, to identify and track a 17-point skeleton on cats. The model is optimized to process video in real-time on low-power edge devices.

To teach the model feline anatomy, a public animal dataset was filtered down to 1,862 images of cats. The model was trained on these images, followed by a second training phase where the data was intentionally distorted by rotating the images, altering lighting, and hiding parts of the subjects. This process forces the model to recognize skeletal structures in challenging conditions, rather than relying on perfect visibility.

Currently, the model struggles with non-standard camera angles, such as a top-down view from a ceiling camera, because the original training data consists primarily of eye-level photos. To resolve this, the next step involves processing custom top-down video footage through the current model to automatically track the skeletons. Any tracking errors will be manually corrected, and this corrected data will be used to retrain the model and improve its performance across diverse perspectives.

The long term goal for this project is to use the fine-tuned model to control a laser pointer to automatically play with my cats.
